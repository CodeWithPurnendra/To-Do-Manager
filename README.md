# ✅ To-Do Manager

A simple command-line **To-Do Manager** built with Python that allows users to save tasks and view them later. This project demonstrates the use of **File Input/Output (File I/O)** for storing tasks permanently in a text file.

## ✨ Features

* ➕ Add new tasks
* 📋 View all saved tasks
* 💾 Save tasks in a text file
* 📂 Persistent storage using File Handling
* 🖥️ Beginner-friendly command-line interface

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/CodeWithPurnendra/To-Do-Manager.git
```

2. Navigate to the project folder:

```bash
cd To-Do-Manager
```

3. Run the application:

```bash 
python To-Do-Manager.py
```

## 🎮 How to Use

1. Start the program.
2. Select an option:

   * **1** → Add a new task.
   * **2** → View all saved tasks.
3. Enter your task when prompted.
4. Your tasks are automatically saved in **tasks.txt** and can be viewed anytime.
